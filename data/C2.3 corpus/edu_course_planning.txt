Course planning agents help students identify the optimal sequence of courses based on academic history, prerequisites, and career goals. A planning agent queries the student transcript, checks prerequisite satisfaction for target courses, detects schedule conflicts, and ranks options by alignment with stated learning objectives. Enrolment actions are irreversible within a registration period, so the agent must request human confirmation before submitting any enrolment request. Machine learning courses typically require linear algebra, probability, and at least one programming language as prerequisites.
